# Prefabs

Reusable model components. Use `"type": "prefab"` with `"src": "prefab-name"`:

```json
{
  "name": "my_chair",
  "type": "prefab",
  "src": "office-chair",
  "position": [2, 0, 0],
  "rotation": [0, 90, 0],
  "scale": [1, 1, 1]
}
```

The authoritative list is the contents of [`../prefabs/`](../prefabs/). The tables below group them for browsing; thumbnails live in [`thumbs/`](thumbs/).
